Lua formatting follows `.stylua.toml`: 2-space indentation, Unix line endings, preferred single quotes when possible, no call parentheses where Stylua can omit them, and a wide 160-column limit. The config follows Kickstart-style readable comments and favors modular plugin specs under `lua/custom/plugins/` while still keeping a large documented `init.lua` baseline. Plugin configuration is declarative through lazy.nvim specs, with many mappings and autocommands defined directly in Lua via `vim.keymap.set` and `vim.api.nvim_create_autocmd`.
