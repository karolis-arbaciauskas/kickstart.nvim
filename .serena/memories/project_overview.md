This repository is a personal fork of nvim-lua/kickstart.nvim for Neovim on Darwin. It is a Lua-based Neovim configuration rooted at init.lua, using lazy.nvim as the plugin manager and lazy-lock.json for pinned plugin versions. Structure: init.lua contains the main Kickstart-derived baseline config and plugin specs; lua/custom/init.lua loads local options, autocmds, and keymaps; lua/custom/plugins/*.lua holds modular plugin overrides/additions; lua/kickstart/plugins/*.lua contains optional Kickstart modules; syntax/ and after/queries/ add custom filetype support (for example Avro IDL and proto queries). The config assumes Neovim 0.11+; kickstart/health.lua explicitly checks for 0.11 and the local machine currently runs 0.11.6.
