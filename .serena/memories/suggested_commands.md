Useful commands for this project: `nvim` to launch the config; `nvim --version` to verify Neovim version; `nvim --headless '+qall'` to validate startup without UI; inside Neovim use `:Lazy` for plugin status and updates, `:Mason` for LSP/tool installation, and `:checkhealth` or `:checkhealth kickstart` for environment diagnostics. Formatting style is governed by `.stylua.toml`; if Stylua is installed, `stylua .` can format Lua files. Git remotes show this repo tracks `origin` as karolis-arbaciauskas/kickstart.nvim and `upstream` as nvim-lua/kickstart.nvim.
